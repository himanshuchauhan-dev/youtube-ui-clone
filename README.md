# YouTube UI Clone

A responsive YouTube UI Clone built using HTML and Tailwind CSS.

## 🚀 Features

- Responsive Layout
- YouTube-like Navigation Bar
- Sidebar Menu
- Video Grid Layout
- Modern UI Design

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
